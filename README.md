📖 Overview

KisanKraft Pro is a modern web-based agricultural machinery booking platform that enables farmers to browse tractor brands, select equipment, estimate costs, apply subsidy calculations, and complete bookings through an intuitive multi-step workflow.

The project demonstrates responsive UI design, dynamic booking management, real-time price calculations, and a professional user experience using pure HTML, CSS, and JavaScript.
